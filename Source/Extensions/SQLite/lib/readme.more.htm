Usage

Install/configure the extensions.
Copy the SQLite.Interop.dll from x86 or x64 directory according to your plateform into the programm directory, or in the website bin directory.
For a "Any CPU" case, copy both directories (x86 and x64) into program directory, or into website root (directory ~/).

Changes from original sources

  • Added overload for SQLiteFunction.RegisterFunction to accept a delegate instead of a type.
  • Added SQLiteConnection.RegisterFunction
